# .NET Notes

This repository is my personal space for learning **.NET** from books.

As I study, I type the code examples myself, keep notes about what I learn, and save small experiments here so I can review them later.

## Purpose

This repo is intended to collect:

- code examples recreated from books,
- notes and short explanations,
- small practice projects,
- experiments with .NET tools and features.

The goal is to keep both the code and the learning notes in one place.

## Workflow

My usual workflow for this repository is:

1. Read a .NET book or chapter.
2. Type the sample code manually to understand it better.
3. Save the code examples and related notes in this repo.
4. Revisit the examples later for practice and review.

## Editor and Tools

- **Editor:** VS Code
- **.NET SDK version:** check it in the terminal with:

```bash
dotnet --version
```

You can also list full SDK details with:

```bash
dotnet --info
```
